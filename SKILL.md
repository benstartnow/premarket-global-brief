# 早间全球要闻 · 开盘前情报层

把一份"开盘前全球市场背景 + A股开盘预判"整理成可发布的 HTML 简报，交付「完整增强版」+「摘要卡」两件套。

## 何时使用
- 用户要"早间要闻 / 开盘前简报 / 晨报 / 盘前情报 / 早盘预判"类内容。
- 需要输出 HTML，且要两份：一份完整详尽、一份浓缩速览。
- 用户给了类似结构的模板，要求按结构产出。

## 核心工作流（务必按顺序）
1. **联网取数（WebSearch）**——用真实数据，不编数。一次并行多组搜索：
   - `US stock market close <昨日日期> Dow Nasdaq S&P 500`、`<昨日日期> sector performance oil Iran`
   - `Nikkei 225 KOSPI open <今日日期> Asian market`
   - `上证指数 深证成指 创业板指 <昨日日期> 收盘 点位 成交额`（中文搜，拿 A股前收）
   - 宏观要闻：地缘、央行纪要、中国 PMI/进出口等
   - 日期推算：脚本用 `date` 取当前日期；"昨日/今日"按交易日历推（周末/节假日顺延）。
2. **生成完整增强版 HTML**（`references/full_version.html` 为黑底科技风模板，红涨绿跌）：
   - 八章结构（含高价值模块）：① 一屏核心结论 ② 隔夜美股全天（三大指数卡 + 11板块强弱表）③ 全球资产温度计（油/金/美元/美债/人民币/VIX + A股映射）④ 外围映射与期指（A50/恒生/中概金龙）⑤ 昨夜预判对账（前日预判 vs 实际，命中率）⑥ 日韩早盘（日经/KOSPI）⑦ 隔夜全球新闻（重大要闻 + A股关联影响）⑧ A股盘面深度（指数/市场宽度/资金流）+ 当日预判（情绪区间 + 板块温度计 + 关注变量）。
   - 遵循**红涨绿跌**（中国习惯）：`.up{color:红} .down{color:绿}`。
   - "预判对账"若无真实前日预判，明确标注为"内部模型回溯示例"，不伪造精确数字。
   - 视觉：黑底（`--bg:#0b0f14`）+ 科技青（`--accent:#2dd4bf`）+ 金色标题（`--gold:#e6b450`）。
3. **生成摘要卡 HTML**（`references/summary_version.html` 为同风格 1-2 屏浓缩模板）：
   - 同一套配色，把八章压成单页：核心结论（4条）+ 外围市场（美股3+日韩2）+ 全球资产温度计（六宫格）+ 外围映射&要闻（双栏）+ A股盘面深度（四宫格+板块标签）+ 当日预判&关注变量（双栏）。
   - 去冗余、不展开表格逐条，只留高价值信息，确保 1-2 屏看完。
   - 板块级表述（如"算力/国资云"），不展开具体股票代码解读。

## 合规红线（通用）
- 不出现"推荐/买入/看好/目标/建仓"等引导词；不引导私信代码、加群、跟车。
- 点位/区间一律标注"情绪参考，非精确预测"。
- 必须带"不荐股、不构成投资建议"声明（页脚/合规条）。
- 若用户明确要发公开平台，再按需补强去个股表述。

## 数据字段清单
- 美股：道指/标普/纳指 收盘点位与涨跌幅、11 行业板块强弱。
- 亚太：日经225、KOSPI 早盘涨跌幅及领涨权重。
- A股前收：上证/深成指/创业板/科创50 点位、涨跌幅、成交额、领涨/领跌板块、市场宽度（红盘家数/涨跌停）、北向资金。
- 全球资产：原油、黄金、美元指数、美债10Y、离岸人民币、VIX。
- 外围映射：富时A50、恒生期货、中概金龙。
- 宏观：地缘事件、央行表态、中国景气/贸易数据。

## 交付物
- `早间全球要闻_<YYYYMMDD>.html`（完整增强版）
- `早间全球要闻_<YYYYMMDD>_摘要.html`（摘要卡）
- 用 present_files 同时交付两个文件。

## 注意事项
- 数据为公开资讯整理，潜在冲突来源以权威源（CNBC/东方财富/新浪）为准，冲突时在文中标注。
- 若用户只要某一版，可只产对应版，不必强求两件套齐发。
- 模板样式可迭代：黑底科技风为主；如用户有品牌色，替换 CSS 变量即可。
